Para poder optimizar más la consulta, se pueden crear índices Para poder buscar rápidamente la fecha de una cita para un paciente

```
CREATE INDEX idx_appointment_patient_date
ON appointment (patient_id, appointment_date);
```

Para poder buscar rápidamente el doctor que atendió a un paciente en una fecha

```
CREATE INDEX idx_mr_patient_date_doctor
ON medical_review (patient_id, appointment_date, doctor_id);
```

Para poder buscar rápidamente la ciudad de un paciente

```
CREATE INDEX idx_patient_city
ON patient (patient_city);
```

Para poder buscar rápidamente la especialidad de un paciente

```
CREATE INDEX idx_doctor_speciality
ON doctor (doctor_speciality);
```
